# stlink-uart
串口和stlink二合一调试工具。

支持的功能：
 * SWIM 编程和调试（STM8）
 * SWD 编程和调试（stm32/ARM系列）
 * UART（导出DTR，支持arduino一键下载）
 * 3.3V提供最大500mA电流，可以作为调试目标的电源输出
 * 电源/tx/rx/stlink指示灯
 * MicroUSB 或者 标准USB A型
